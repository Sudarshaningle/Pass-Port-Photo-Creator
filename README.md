# pic.io — Passport Photo Studio

A browser-based passport photo generator that creates print-ready PDFs with precise sizing, layout, and DPI control.

## Features
- Multiple passport standards (US, India, EU, China)
- Auto grid layout calculation
- Adjustable margins and spacing
- Background color customization
- 300 / 600 DPI export
- Cut lines for printing
- High-quality PDF generation (jsPDF)

## Usage
1. Upload photo
2. Select standard and paper
3. Adjust settings
4. Generate PDF

## Tech Stack
- Vanilla JavaScript
- HTML5 Canvas
- jsPDF
